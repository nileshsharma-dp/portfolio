    Browse Through My

    Projects

    Cyclistic Bike-Share Analytics

    A comprehensive analysis of Cyclistic Bike-Share program, featuring Exploratory Data Analysis (EDA), A/B Testing, SARIMA Time Series Forecasting and interactive Tableau dashboards to uncover user behavior, trends and marketing strategies for growth.

    Bank Marketing Campaign Analysis

    This project focuses on analyzing a bank's marketing campaign data using Python. It involves data preprocessing, exploratory data analysis, and the application of machine learning models to predict customer responses to marketing efforts. The goal is to identify key factors influencing customer decisions and enhance the effectiveness of future campaigns.

    Marques Brownlee YouTube Analysis

    An in-depth exploration of MKBHD's YouTube journey, utilizing the YouTube API for conducting Exploratory Data Analysis (EDA), Natural Language Processing (NLP) and sentiment analysis through VADER and a Multinomial Naive Bayes Classifier. Provides insights and recommendations for content creators in the digital landscape.

    Football Players Clustering Analysis

    Exploration of football players' clustering analysis with KMeans using FIFA 23 data to discover insights and trends in the world of sports.

    Customer Churn Analytics

    An endeavor in customer churn prediction through logistic regression, harnessing data science for enhanced customer retention.
